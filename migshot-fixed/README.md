## Installation

1. Download and extract the zip file
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the extracted folder
6. **Important**: Reload any open Instagram tabs after installing

## Usage

1. Navigate to ANY website
2. Click the MIGShot extension icon
3. **(Optional)** Check "üìÑ Mark as About Page" if capturing an About/profile page
4. Click "üì¶ Archive This Post" (or press **Alt+S**)
5. **Click and drag** to select the area you want to capture
6. Release to capture (or press ESC to cancel)
7. View your captures in the Archive
8. **Use tabs** to filter by platform

### Tips
- **Alt+S** = instant capture (just like Windows Snipping Tool!)
- Works on Facebook, Instagram, TikTok, Twitter, Reddit, LinkedIn, YouTube, and ANY website!
- Archive automatically creates tabs for each platform
- **Mark About Pages** to easily identify profile/business information captures
- Click "Copy All Captures" to copy only the current tab's captures
- Your profile picture and comment input are automatically hidden (on supported platforms)
- Minimum selection size: 50x50 pixels
- Archive cards are collapsible for easier browsing
